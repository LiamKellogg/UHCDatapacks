# UHCDatapacks

This repo contains 3 datapacks, each plays a part in the UHC game im trying to do:
 - uhchandler: This manages the players, deathmatch, progression
 - uhcrecipies: This has the custom crafts
 - noaxeshield, should we want to play a more hypixelesque version of UHC

There is the Lab Notes folder, where I keep notes of things I would like to add and how I would add them
The Last folder is autogenrecipes, which is to try and automate the process of creating the custom recipes
